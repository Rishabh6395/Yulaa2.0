1. System Overview

The School Management System is a multi-tenant web application that covers:

Leave Management

Submit, approve, and track all leave types with multi-step workflow.

Attendance

Daily punch-in/out, subject attendance for students, monthly reports.

Timetable

Class schedules, period management, substitute teacher assignments.

Scheduling

Teacher reassignments, conflict detection, automatic notifications.

Holiday Calendar

School holidays, optional/mandatory types, weekoff configuration.

Notifications

In-app and push alerts for all key events.

Every user belongs to exactly one school. Data is strictly isolated — no user can see or modify data from another school.

3. Super Admin

super_admin

The Super Admin has system-wide access across all schools. This role is typically held by the platform owner.

School Management

  • Navigate to Dashboard → Schools
  • Click Add School and fill in school name, address, contact details
  • After creating the school, assign a Principal from the staff list
  • Configure the school's Leave Policy under the school settings page

Leave Configuration per School

Go to Schools → [School Name] → Leave Config:

  • Leave Types — create types (e.g., Sick Leave, Casual Leave) with role applicability
  • Balance Policies — set days per year, carry-forward rules per role
  • Approval Workflow — define multi-step approval chains for parent and teacher leaves
  • Weekoffs — select which days (Sun–Sat) are non-working for this school
  • Holidays — add individual holidays or bulk upload via Excel/CSV
Holiday configuration requires either Super Admin or Principal role. School Admin cannot edit holidays.

Holiday Bulk Upload (Excel/CSV)

  • Download the template: click Download Template in the Holiday section
  • Fill in columns: Date (YYYY-MM-DD), Holiday Name, Type (mandatory/optional)
  • Upload the filled file and confirm
Uploading a file will upsert (add or update) holidays. Existing holidays with the same date will be updated, not duplicated.

4. School Admin

school_admin

Manages day-to-day school operations within their assigned school.

Staff Management

  • Add teachers and non-teaching employees via Staff → Add Member
  • Assign subjects and classes to teachers
  • Deactivate staff accounts when they leave

Class & Section Setup

  • Create classes (Grade 1 – Grade 12) with sections (A, B, C…)
  • Enrol students to classes via Students → Assign Class

Leave Policy Management

  • Create leave types for different roles (teacher, employee)
  • Set annual balance and carry-forward rules
  • Define multi-step approval workflows
School Admins cannot add or edit holidays. Only Super Admin and Principal can manage the holiday calendar.

Timetable Upload

  • Go to Schools → [School] → Timetable
  • Download the Excel template
  • Fill in class_id, day_of_week (0=Sun…6=Sat), period_no, start_time, end_time, subject, teacher_id
  • Upload and confirm. Existing slots for the class are replaced.

Leave Approval (proxy)

School Admin can approve leaves on behalf of any step in the workflow and can create teacher reassignments as a proxy.

5. Principal

principal

Holiday & Weekoff Configuration

  • Go to Dashboard → Leave Config → Holidays
  • Click Add Holiday, enter date, name, type (mandatory/optional)
  • Or upload an Excel file for bulk import
  • To set weekoffs: click Weekoffs tab and select days
Holidays are linked to the current academic year automatically. The system uses the Indian academic year (April to March). May 2026 belongs to academic year 2026-2027.

Leave Approvals

The Principal typically acts as the final approver in the leave workflow.

  • View pending approvals under Leave → Pending
  • Click a leave request to see details, then Approve or Reject
  • Approved leaves automatically deduct from the employee's leave balance

Reports

  • Monthly attendance summary for all staff
  • Leave balance reports per employee
  • Class-wise timetable overview

6. Head of Department (HOD)

hod

Department Leave Approvals

HODs are typically configured as a mid-step reviewer in the leave approval workflow for their department.

  • Receives notification when a teacher in their department submits leave
  • Reviews and approves/rejects at their designated step
  • Cannot skip steps or approve on behalf of another step

Timetable View

  • View all class timetables under their department
  • See active reassignments for department teachers
  • Cannot modify timetables (read-only unless also Principal/Admin)

7. Teacher

teacher

Daily Timetable

  • Go to Dashboard → Timetable to view today's schedule
  • Slots are colour-coded: Assigned (your normal class), Substitute (covering for a colleague), Reassigned Away (your class is being covered)
  • Use the date picker to view any day's schedule

Marking Student Attendance

  • Go to Dashboard → Attendance
  • Select the class and date
  • For each student, select: Present / Absent / Late / Half Day / Excused
  • Click Save. Attendance can be edited until end of day.

Subject-wise Attendance (for Exam/Report purposes)

  • Available for classes where a timetable is configured
  • Each period's attendance is recorded independently per subject

Submitting Leave

  • Go to Dashboard → Leave → Apply
  • Select leave type, start date, end date, and reason
  • The system shows Effective Days (excluding weekoffs and holidays)
  • Submit — the request enters the approval workflow

Teacher Reassignment (substitute request)

  • Go to Dashboard → Timetable → Reassign
  • Select the slot (period/subject) you want to hand off
  • Choose a substitute teacher from your school
  • Set the date range and add a reason (e.g., "on leave")
  • Submit — the substitute is notified automatically
You cannot assign a substitute teacher who has approved leave during the selected dates. The system will block the request.

Withdrawing Leave

Pending or approved leave can be withdrawn from Leave → My Requests. When you withdraw approved leave:

  • Any timetable reassignments created for the leave period are automatically cancelled
  • The substitute teacher receives a cancellation notification

Own Attendance

  • View monthly punch-in/out record under Attendance → My Attendance
  • See working hours, status (present/late/absent), holiday and leave overlay

8. Employee (Non-Teaching Staff)

employee

Leave Application

  • Go to Dashboard → Leave → Apply
  • Choose leave type (Sick, Casual, Other — as configured by admin)
  • Pick dates — the system shows effective working days and remaining balance
  • Submit; the request routes through the configured approval workflow

Leave Balance

  • View remaining leave balance on the Leave dashboard
  • Balance is deducted automatically when leave is finally approved
  • Carry-forward balance (if enabled) is shown separately

Attendance

  • Monthly attendance report available under Attendance → My Attendance
  • Shows punch-in, punch-out, working hours, and attendance rate
  • Approved leave days are shown as "Leave" (not Absent)

9. Parent

parent

Switching Between Children

If you have multiple children enrolled, use the child selector at the top of the Leave and Attendance pages to switch context. Data shown always belongs to the currently selected child only.

Submitting Leave for Your Child

  • Go to Dashboard → Leave
  • Select the child from the dropdown
  • Click Apply Leave, choose type (Sick, Emergency, Other), dates, and reason
  • Effective days are calculated excluding the child's class weekoffs and school holidays
  • Submit — you will be notified when the leave is approved or rejected
The school may configure a custom approval workflow. Your leave request will show the current step and approver at each stage.

Viewing Your Child's Attendance

  • Go to Dashboard → Attendance and select your child
  • See daily status, subject-wise attendance, and monthly summary
  • Holiday and weekoff days are clearly marked

School Holiday Calendar

  • Go to Dashboard → Calendar
  • Holidays are shown in orange; your child's class weekoff days are grayed out
  • Weekoff days are derived from the class timetable (which days have no classes)
You can only access data for children linked to your account. Attempting to view another student's data is blocked by the system.

student

Viewing Attendance

Leave Status

Holiday Calendar

11. Leave Management — Deep Dive

Leave Lifecycle

StatusMeaningNext Action
PendingSubmitted, awaiting reviewReviewer approves/rejects at current step
ApprovedAll steps passedBalance deducted; attendance synced for students
RejectedRejected at some stepApplicant notified; no balance change
WithdrawnCancelled by applicantBalance restored; reassignments cancelled

Effective Day Calculation

When applying for leave, the system calculates Effective Days — the actual number of working days, excluding:

  • School weekoff days (configured per school)
  • Public holidays in the school's holiday calendar
  • For student leaves: class timetable weekoff days (days with no periods)
A leave application entirely on weekoffs/holidays is blocked with an error message.

Multi-Step Approval Workflow

School admins can configure up to N steps for parent and teacher leaves. Each step specifies:

  • Step Order — sequence number (1, 2, 3…)
  • Approver Role — the role that can review this step (e.g., hod, principal)
  • Specific Approver — optionally lock the step to one person
Only the correct role/user can approve each step. Trying to approve out of order or out of role will be rejected.

Leave Balance

  • Balances are set per-school per-role per-leave-type
  • Annual reset happens at the start of each academic year (April 1)
  • Carry-forward is optional and capped by maxCarryDays
  • Deduction happens only when the leave reaches fully approved status

Overlap Protection

You cannot apply for leave on dates that overlap with an existing pending or approved leave for the same person/student.

Leave & Attendance Sync (Students)

When a student's leave is approved, the system automatically marks their attendance as "Leave" for each effective day. If the leave is later withdrawn, these attendance records are deleted.

Monthly Attendance Report

Available at Attendance → Monthly. The report shows:

  • Day-by-day status for the selected month
  • Punch-in / Punch-out times and working hours
  • Summary: Present, Absent, Late, Half Day, Leave, Holidays, Weekoffs
  • Attendance Rate = (Present + Late + Half Day + Excused) / Working Days × 100

Bulk Student Attendance Upload

Teachers can upload a CSV for bulk student attendance marking:

  • Download the CSV template from the Attendance page
  • Fill in student_id and status for each row
  • Upload — the system validates that every student_id belongs to the selected class and school
  • Invalid rows are skipped with a count reported back
Students from other classes or schools in the CSV are silently skipped. They cannot be accidentally recorded.

Academic Year Handling

The system uses the Indian academic year (April–March). When fetching attendance for January 2026, the academic year is automatically computed as 2025-2026, and for May 2026 it becomes 2026-2027. This ensures correct holiday lookup.

13. Timetable & Scheduling — Deep Dive

Creating a Timetable (Admin)

  • Go to Schools → [School] → Timetable
  • Select a class and click Edit Timetable
  • Add periods: day, period number, start/end time, subject, teacher
  • Save — the timetable is stored for the current academic year
  • Alternatively, upload an Excel/CSV in bulk

Teacher Daily View

Each teacher sees three types of slots for a given date:

Slot TypeColourMeaning
AssignedBlueYour regular class for this period
SubstituteGreenYou are covering for a colleague
Reassigned AwayGrayYour class is being covered by someone else

Teacher Reassignment

A reassignment lets Teacher A hand off a specific period to Teacher B for a date range.

  • Overlap check — cannot create a second reassignment for the same slot and overlapping dates
  • Leave conflict check — substitute must not have approved leave during the dates
  • Auto-notification — substitute is notified immediately when assigned
  • Cancellation notification — substitute is notified if the reassignment is cancelled

Leave → Reassignment Cascade

When a teacher withdraws an approved leave, the system:

  1. Marks all related reassignments (created during the leave period) as inactive
  2. Notifies each substitute teacher of the cancellation

Student Weekoff via Timetable

A student's weekoff days are derived from their class timetable — any day with no periods is treated as a weekoff. This is used when calculating effective leave days and showing the holiday calendar. If no timetable exists, the school's leave-config weekoff is used as a fallback.

14. Holidays & Weekoffs

How Weekoffs are Stored

Weekoff days are stored as special entries in the Holiday Calendar with academicYear = '_weekoff_'. Each weekday (Sun–Sat) maps to a canonical 1970 epoch date:

DayEpoch Date
Sunday (0)1970-01-04
Monday (1)1970-01-05
Tuesday (2)1970-01-06
Wednesday (3)1970-01-07
Thursday (4)1970-01-08
Friday (5)1970-01-09
Saturday (6)1970-01-10

Adding Holidays

  • Single holiday: Holiday name, date, type (mandatory/optional)
  • Bulk upload: Excel/CSV with Date, Name, Type columns
  • Uploading a holiday that already exists on the same date will update it (not duplicate)

Academic Year and Holiday Lookup

Holidays are scoped by academic year. The system automatically computes the academic year for any date:

  • April 2026 – March 2027 → 2026-2027
  • January 2026 – March 2026 → 2025-2026
From version 1.0 onwards, the academic year is computed dynamically. You no longer need to manually set the year — holidays added in May 2026 are automatically stored under 2026-2027.

16. FAQ & Troubleshooting

Why are my holidays not showing in the calendar?

Holidays are stored per academic year. If you added a holiday in May 2026 but selected academic year 2025-2026 in the admin panel, the holiday was stored under 2026-2027 and the calendar (which fetches the correct year) will show it correctly. Make sure the year selector in the Leave Config panel matches the year you intend to edit.

Why is my effective leave count different from the calendar days?

Effective days exclude weekoffs and school holidays. If you select 5 calendar days but 2 of them are weekoffs, you'll see 3 effective days. The leave balance will only be deducted for the effective days.

I see leaves from another class/school in the list

This should not happen. All API endpoints scope data to the authenticated user's school. If you see foreign data, please report it immediately to your system administrator.

Why can't I approve a leave request?

The approval workflow has steps. You can only approve at the step assigned to your role. If the leave is at step 1 (HOD) and you are the Principal (step 2), you must wait for the HOD to complete their step first.

I withdrew my leave but my substitute wasn't notified

Notifications are sent automatically when a leave is withdrawn. If the substitute didn't receive a notification, check that they have enabled browser push notifications and that their account is active.

The timetable shows wrong day for a date

The system uses UTC date parsing to avoid timezone drift. All dates are parsed as UTC midnight. If you see a one-day offset, ensure dates are entered in YYYY-MM-DD format without timezone suffixes.

How do I add more academic years to the year selector?

Contact your system administrator. Academic years are managed in the code configuration and can be extended. Currently supported: 2024-2025, 2025-2026, 2026-2027, 2027-2028.

A substitute teacher was assigned even though they are on leave

From version 1.0 onwards the system blocks reassignment if the substitute has an approved leave during the selected dates. If this happened with an older version, please cancel the reassignment manually.
